JS Language

Parentheses are used to set a function's parameters but they can also be blank if no specific parameters are to be set and the function is to just run as it is.

Square Brackets are used for arrays, to show the start and end point of the array items. These can be nested inside each other

Braces are used to contain the components of a function. I guess that means For and If statements are a kind of function since they have parameters and braces?

Single quotes are used to contain strings or arguments.

Double quotes are used to contain strings or arguments also. It helps to be able to use both to differentiate between the two.

It's good to know now that they are called 'braces' and not 'squiggly brackets'.
