Solving Problems

I assume this question is relating to code but I haven't actually solved any coding problems in an elegant way. So instead I've only got a film example where we had to shoot with a window in the background but the light from the window was blowing out the highlights and making it difficult to correctly expose the entire shot, but closing the window meant there wasn't enough light. So what I did was filmed the footage with the areas not near the window exposed properly, and let the highlights clip so the window was glaringly white. I then doubled the footage onto itself and created a mask over the window to exposure correct the window only while leaving the rest of the shot at its normal exposure.
