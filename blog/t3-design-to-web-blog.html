Technical Reflections pt.3

A responsive site is one that adjusts itself based on the screen size of the device that it is being viewed on. This is important because nowadays people are viewing things on a range of devices and I'm pretty sure this is basically exactly the same question as the previous technical blog. Mobile first design is making sure the site is functional from mobile devices first then building up from there. This is important because mobile functionality is generally the hardest to do, so getting this sorted first means you don't get it sorted for desktop then have to tweak it for mobile constantly.
